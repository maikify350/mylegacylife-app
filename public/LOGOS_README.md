# Logo Files

This directory contains theme-specific logo variations:

## Logo Variants

- `logo_mocha.png` - Mocha Mousse theme (default, light background)
  - Colors: Mocha brown (#A47764), Deep teal (#2D5F5D)
  - Best on warm cream background (#FDF8F5)

- `logo_dark.png` - Dark theme
  - Colors: Lighter teal (#14B8A6), adjusted for dark background
  - Best on dark slate background (#0F172A)

- `logo_light.png` - Light teal theme
  - Colors: Teal (#0F766E)
  - Best on light slate background (#F8FAFC)

## Usage

The app automatically selects the appropriate logo based on the active theme.

## File Requirements

- Format: PNG with transparent background
- Recommended size: 512x512px or larger
- Transparent background required
- High quality for retina displays
